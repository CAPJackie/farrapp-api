## Get Started

To get started consider:

### Set up project

- install dependencies:

  ```bash
  gradlew dependencies
  ```

- run application locally:

  ```bash
  gradlew bootRun
  ```

- run tests:

  ```bash
  gradlew test
  ```

- create a build:
  ```bash
  gradlew build
  ```
